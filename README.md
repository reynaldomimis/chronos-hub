# Chronos Hub: Rewarding games

**Chronos Hub** is a centralized rewards-based application that combines multiple casual and skill-based games into one platform. Users can play games, complete simple tasks, and participate in daily activities to earn virtual rewards — **no real money is involved at any stage of the system**.

All rewards can be converted using in-app coins into items such as mobile load, digital rewards, diamonds, and other available in-app benefits, which users can claim and use personally depending on what is offered inside the platform.

---

## Overview

Chronos Hub: is built as an entertainment and engagement platform where users earn points or coins through gameplay and activities. These rewards are strictly virtual and are used only within the app for redemption purposes.

The platform focuses on:
- Fun and accessible games
- Simple and fair reward mechanics
- No financial risk or monetary involvement
- Transparent and user-friendly experience
